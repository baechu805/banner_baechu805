# show-banner

`show-banner`는 입력된 텍스트를 `figlet`을 사용하여 배너 형식으로 출력하는 간단한 명령어입니다.

### 요구 사항

- `figlet` 설치가 필요합니다. `figlet`은 텍스트를 멋진 아스키 아트 배너로 변환하는 도구입니다.


#### Ubuntu/Debian

```sh
sudo apt-get update
sudo apt-get install figlet
